Focusrite


Focusrite not appearing in software

Step 1

Power Cylce the Focusrite. Make sure the software is running before power cycling.

Step 2

Restart the application


  1. Press CTRL + SHIFT + ESC
  2. In the "Processes" tab, find and right click the "Focusrite Control" application
  3. Select the "End Task"
  4. Open the application once more

Step 3

Switch the USB-C cable


Note: Some USB-C cables are power only and not data transfer + power.

Step 2
